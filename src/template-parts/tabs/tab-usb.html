USB route:

USB Modes:
  • Host: Allows peripherals (such as keyboards, mice, USB drives, etc.) to be connected to the selected node via the USB_OTG port;
  • Device: Allows the selected node to be connected as a USB device to an external USB host via the USB_OTG port;
  • Flash: Allows the selected node to be connected as a USB device to an external USB host via the USB_OTG port and also set the mode of the node to allow it to be flashed.
